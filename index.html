Namagame Creator 配布ページ

ニコ生ゲームをテキストから生成し、実行確認と配布までを行えるデスクトップアプリです。

アプリをダウンロードする

Windows向け
win-unpacked.zip をダウンロード
Linux向け
linux-unpacked.zip をダウンロード

ソースからビルドして利用する

ダウンロード済みバイナリではなく自分でビルドする場合は、以下の手順で実行します。

git clone https://github.com/dera-/namagame-creator-app-win.git
cd namagame-creator-app-win
git submodule update --init --recursive
npm install
cd akashic-mcp
npm install
cd ..
cd playground
npm install
npm run build
cd ..
npm run build
npm start

詳細な説明や最新情報は README.md を参照してください。

利用方法と画面説明

1. API設定画面

利用するモデルとAPIキーを設定して、アプリ全体で使う接続情報を保存します。

2. ゲーム生成画面

作りたいゲームの内容を入力して生成できます。既存プロジェクトの読み込みにも対応しています。

3. ゲーム実行画面

生成したゲームをその場で実行して確認します。必要に応じて修正と再実行を繰り返せます。

生成サンプルゲーム

本アプリで生成したサンプルゲームをブラウザで確認できます。
